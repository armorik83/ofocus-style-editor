# ofocus-style-editor
OmniFocus 2.5+ Style plist editor

# How to use

- Clone this.
- `npm i`
- `npm run build`
- `npm start`
- Open http://127.0.0.1:8080 in your browser.
- Click [Choose file] -> [Import]
- Anything you like.
- Click [Export]
- Show the console and copy / paste.
- Import in OmniFocus.
